## How to run (example)
1. Clone repo
2. Create virtual env: `python -m venv venv` or `conda env create -f environment.yml`
3. Install: `pip install -r requirements.txt`
4. Run demo notebook: `jupyter lab notebooks/01-demo.ipynb`
